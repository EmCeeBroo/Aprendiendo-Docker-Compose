# 🐳 Aprendiendo Docker Compose

Training material developed and delivered as a knowledge-sharing session for software development students at **SENA (ETP)**.

This repository covers Docker fundamentals through practical service orchestration with Docker Compose, including networking, volumes, and recommended next steps for continued learning.

## 🚀 Suggested Reading Order

If you're new here, follow the pages in order: start with the concepts in **page 1**, apply them in the **practice exercise**, then check the **next steps** and **additional resources** to keep growing.

---

## 🛠️ Prerequisites

To follow along with the exercises you'll need:

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed (includes Docker Compose)
- Basic familiarity with the terminal / command line
